# 故障排查指南

## 数据库相关

### 问题1：无法连接到MySQL数据库

**错误信息**:
```
Can't connect to MySQL server on 'localhost'
```

**可能原因**:
1. MySQL服务未启动
2. 数据库配置错误
3. 防火墙阻止连接

**解决方法**:
```bash
# 检查MySQL服务状态
net start MySQL

# 检查配置文件 backend/.env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=chatroom
```

---

### 问题2：表不存在

**错误信息**:
```
Table 'chatroom.users' doesn't exist
```

**解决方法**:
```bash
# 初始化数据库
cd backend
python app.py init-db
```

---

## 后端相关

### 问题3：模块导入错误

**错误信息**:
```
ModuleNotFoundError: No module named 'flask'
```

**解决方法**:
```bash
cd backend
pip install -r requirements.txt
```

---

### 问题4：端口被占用

**错误信息**:
```
Address already in use
```

**解决方法**:
```bash
# 查找占用端口的进程
netstat -ano | findstr :5000

# 结束进程
taskkill /PID <进程ID> /F

# 或修改端口
# 在 backend/.env 中修改 PORT=5001
```

---

## 前端相关

### 问题5：无法连接到服务器

**错误信息**:
```
无法连接到服务器
```

**解决方法**:
1. 确认后端服务已启动
2. 检查 `frontend/config.py`:
```python
SERVER_HOST = 'http://localhost:5000'
```

---

### 问题6：Token无效

**错误信息**:
```
Token无效或已过期
```

**解决方法**:
1. 重新登录
2. 或删除Token文件: `C:\Users\你的用户名\.chatroom\token.txt`

---

## WebSocket相关

### 问题7：消息无法发送

**可能原因**:
1. WebSocket未连接
2. 未认证
3. 网络问题

**解决方法**:
1. 检查状态栏是否显示"在线"
2. 重启客户端
3. 检查后端日志

---

## 通用问题

### 问题8：Python版本不兼容

**错误信息**:
```
SyntaxError: invalid syntax
```

**解决方法**:
```bash
# 检查Python版本（需要3.8+）
python --version

# 如果版本过低，请升级Python
```

---

### 问题9：依赖包版本冲突

**解决方法**:
```bash
# 创建虚拟环境
python -m venv venv
venv\Scripts\activate

# 重新安装依赖
pip install -r requirements.txt
```
