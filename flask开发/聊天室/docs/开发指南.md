## 后端开发

### 添加新的数据模型

在 `backend/models/` 创建模型文件：

```python
from datetime import datetime
from .user import db

class YourModel(db.Model):
    __tablename__ = 'your_table'
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(50), nullable=False)
    created_at = db.Column(db.DateTime, default=datetime.utcnow)
```

### 添加新的API接口

在 `backend/routes/` 创建路由文件：

```python
from flask import Blueprint, request
from utils import token_required, success_response

your_bp = Blueprint('your_route', __name__)

@your_bp.route('', methods=['GET'])
@token_required
def get_list():
    return success_response(data=[], message='成功')
```

### 添加WebSocket事件

在 `backend/sockets/chat.py` 添加：

```python
@socketio.on('your_event')
def handle_your_event(data):
    emit('response_event', {'result': 'success'})
```

## 前端开发

### 创建新窗口

在 `frontend/ui/` 创建窗口文件：

```python
from PySide6.QtWidgets import QWidget, QVBoxLayout

class YourWindow(QWidget):
    def __init__(self):
        super().__init__()
        self.init_ui()
```

### 添加API调用

在 `frontend/network/api_client.py` 添加：

```python
def your_method(self):
    return self._request('GET', '/api/your_route')
```

## 代码规范

### 命名规范
- 类名：`UserModel`
- 函数：`get_user_info`
- 变量：`user_name`
- 常量：`MAX_LENGTH`

### 注释规范
- 所有类和函数都要有文档字符串
- 使用中文注释
- 复杂逻辑添加行内注释
