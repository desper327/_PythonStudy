# Flask-PySide6 聊天室系统 💬

## 📝 项目简介

这是一个基于 **Flask** 后端和 **PySide6** 桌面客户端的实时聊天室应用，支持用户私聊和群组聊天功能。

> ⚡ **项目特点**: 现代化UI设计 · 实时消息推送 · 完整注释 · 易于扩展

## ✨ 主要特性

- 🔐 **用户管理**: 注册、登录、个人信息管理
- 💬 **私聊功能**: 一对一实时消息通信
- 👥 **群组聊天**: 创建群组、加入群组、群组消息
- 📨 **实时通信**: 基于 WebSocket 的实时消息推送
- 💾 **消息持久化**: MySQL 数据库存储所有消息
- 🎨 **现代化UI**: 基于 PySide6/Qt 的桌面客户端

## 🛠️ 技术栈

### 后端
- **框架**: Flask 2.3.0
- **实时通信**: Flask-SocketIO 5.3.4
- **数据库**: MySQL + SQLAlchemy ORM
- **认证**: JWT (JSON Web Token)
- **密码加密**: bcrypt

### 前端
- **UI框架**: PySide6 6.5.0
- **网络通信**: 
  - REST API: requests
  - WebSocket: python-socketio[client]

## 🚀 一键启动

### 环境要求
- Python 3.8+
- MySQL 5.7+ / MySQL 8.0+

### Windows 快速启动

1. **配置数据库**
   - 编辑 `backend/.env` 文件，设置MySQL密码
   ```ini
   DB_PASSWORD=你的MySQL密码
   ```

2. **初始化数据库**（首次运行）
   ```bash
   双击运行: 初始化数据库.bat
   ```

3. **创建测试数据**（可选）
   ```bash
   双击运行: 创建测试数据.bat
   ```
   会创建以下测试账号：
   - admin / 123456
   - user1 / 123456
   - user2 / 123456
   - user3 / 123456

4. **启动后端服务器**
   ```bash
   双击运行: 启动后端.bat
   ```

5. **启动前端客户端**（新窗口）
   ```bash
   双击运行: 启动前端.bat
   ```

### 手动启动

#### 后端
```bash
cd backend
pip install -r requirements.txt
python app.py init-db  # 首次运行
python app.py
```

#### 前端
```bash
cd frontend
pip install -r requirements.txt
python main.py
```
