# 快速开始指南

## 📋 前置要求

- Python 3.8+
- MySQL 5.7+ 或 MySQL 8.0+
- Windows操作系统

## 🚀 安装步骤

### 1. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 配置数据库

#### 创建数据库

```sql
mysql -u root -p
CREATE DATABASE chatroom CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit
```

#### 配置环境变量

复制 `.env.example` 文件并重命名为 `.env`，然后修改数据库配置：

```bash
cp .env.example .env
```

编辑 `.env` 文件：

```ini
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的MySQL密码
DB_NAME=chatroom

# JWT密钥（生产环境请改为随机字符串）
JWT_SECRET_KEY=your-very-secure-random-secret-key
SECRET_KEY=your-flask-secret-key
```

#### 初始化数据库表

```bash
cd backend
python app.py init-db
```

看到 "数据库初始化成功！" 表示初始化完成。

### 3. 安装前端依赖

```bash
cd frontend
pip install -r requirements.txt
```

### 4. 启动服务

#### 启动后端服务器

```bash
cd backend
python app.py
```

后端服务将运行在 `http://localhost:5000`

#### 启动前端客户端

打开新的终端窗口：

```bash
cd frontend
python main.py
```

## 🎯 使用说明

### 首次使用

1. 启动前端客户端后会看到登录界面
2. 点击"还没有账号？立即注册"
3. 填写注册信息：
   - 用户名：3-50个字符，只能包含字母和数字
   - 昵称：显示名称
   - 邮箱：可选
   - 密码：至少6个字符
4. 注册成功后自动返回登录界面
5. 使用注册的用户名和密码登录

### 功能使用

#### 搜索用户（私聊）
1. 在左侧"联系人"标签页
2. 输入用户名或昵称关键词
3. 点击"搜索"
4. 点击搜索结果中的用户开始聊天

#### 创建群组
1. 菜单栏 -> 群组 -> 创建群组
2. 输入群组名称
3. 创建成功后自动出现在"群组"列表

#### 加入群组
1. 菜单栏 -> 群组 -> 加入群组
2. 选择要加入的公开群组
3. 加入成功后可以在群组列表看到

#### 发送消息
1. 选择联系人或群组
2. 在底部输入框输入消息
3. 点击"发送"或按 Ctrl+Enter

#### 查看群组成员
1. 在群组列表中右键点击群组
2. 选择"查看成员"

#### 退出群组
1. 在群组列表中右键点击群组
2. 选择"退出群组"
3. 确认退出

## ⚙️ 配置说明

### 后端配置 (backend/config.py)

- `DB_HOST`: 数据库地址
- `DB_PORT`: 数据库端口
- `DB_USER`: 数据库用户名
- `DB_PASSWORD`: 数据库密码
- `DB_NAME`: 数据库名称
- `JWT_EXPIRE_HOURS`: Token过期时间（小时）
- `HOST`: 服务器监听地址
- `PORT`: 服务器端口

### 前端配置 (frontend/config.py)

- `SERVER_HOST`: 后端服务器地址（默认：http://localhost:5000）
- `SOCKET_HOST`: WebSocket服务器地址（默认：http://localhost:5000）
- `WINDOW_WIDTH`: 窗口宽度
- `WINDOW_HEIGHT`: 窗口高度

## 🔧 常见问题

### 1. 无法连接到服务器

**原因**：后端服务未启动或配置错误

**解决**：
- 确认后端服务正在运行
- 检查前端 `config.py` 中的 `SERVER_HOST` 配置
- 检查防火墙设置

### 2. 数据库连接失败

**原因**：数据库配置错误或MySQL服务未启动

**解决**：
- 确认MySQL服务正在运行
- 检查 `.env` 文件中的数据库配置
- 确认数据库已创建

### 3. 导入模块错误

**原因**：依赖包未安装

**解决**：
```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
pip install -r requirements.txt
```

### 4. Token无效或已过期

**原因**：Token过期（默认24小时）

**解决**：
- 重新登录
- 或删除本地Token文件：`C:\Users\你的用户名\.chatroom\token.txt`

### 5. 消息发送失败

**原因**：WebSocket连接断开

**解决**：
- 检查网络连接
- 检查后端服务是否正常
- 重启客户端
