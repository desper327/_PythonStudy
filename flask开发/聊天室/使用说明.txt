【快速开始】

1. 配置数据库
   - 打开 backend\.env 文件
   - 修改 DB_PASSWORD 为你的MySQL密码

2. 初始化数据库（首次运行）
   - 双击运行: 初始化数据库.bat

3. 创建测试数据（可选）
   - 双击运行: 创建测试数据.bat
   - 会自动创建4个测试账号

4. 启动后端服务器
   - 双击运行: 启动后端.bat
   - 看到 "Running on http://..." 表示启动成功

5. 启动前端客户端
   - 双击运行: 启动前端.bat
   - 登录窗口会自动打开

【功能说明】

◆ 用户注册
  - 点击"还没有账号？立即注册"
  - 填写用户名、昵称、密码
  - 注册成功后自动返回登录

◆ 搜索用户（私聊）
  - 切换到"联系人"标签页
  - 输入用户名或昵称
  - 点击搜索结果开始聊天

◆ 创建群组
  - 菜单栏 -> 群组 -> 创建群组
  - 输入群组名称
  - 创建成功后出现在群组列表

◆ 加入群组
  - 菜单栏 -> 群组 -> 加入群组
  - 选择要加入的公开群组
  - 加入成功后可以聊天

◆ 发送消息
  - 选择联系人或群组
  - 在底部输入框输入
  - 点击"发送"或按 Ctrl+Enter

◆ 查看群组成员
  - 在群组列表右键点击群组
  - 选择"查看成员"

◆ 退出群组
  - 在群组列表右键点击群组
  - 选择"退出群组"

【常见问题】

问: 无法连接到服务器？
答: 请确保后端服务已启动（运行"启动后端.bat"）

问: 数据库连接失败？
答: 检查MySQL是否运行，密码是否正确（backend\.env）

问: Token无效？
答: 重新登录或删除 C:\Users\你的用户名\.chatroom\token.txt

问: 依赖包安装失败？
答: 尝试升级pip: python -m pip install --upgrade pip
