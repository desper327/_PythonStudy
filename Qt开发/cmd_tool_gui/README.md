# 命令行工具GUI

一个基于PySide6的现代化GUI工具，集成了各种常用命令行工具，包括UV包管理器、网络命令、操作系统命令和系统命令。

## 功能特点

- 标签页式界面，方便切换不同类型的命令
- 美观现代的用户界面
- 命令历史记录和预定义命令列表
- 实时命令输出显示
- 自定义命令保存功能
- 支持停止正在运行的命令
- 暗色/亮色主题切换
- 自定义命令的导入/导出功能
- 配置文件自动保存
- 用户反馈收集系统

## 安装

1. 克隆或下载此仓库
2. 安装依赖: `pip install -r requirements.txt`
3. 运行程序: `python cmd_tool_gui.py`

## 使用方法

1. 在界面顶部的标签页中选择需要使用的命令类别(UV包管理器、网络命令、OS命令、系统命令或自定义命令)
2. 从下拉菜单中选择预定义命令或直接在输入框中输入自定义命令
3. 点击"运行"按钮执行命令
4. 命令输出将显示在文本区域
5. 使用"清除"按钮清空输出区域
6. 如需停止正在运行的命令，点击"停止"按钮

## 自定义命令

在"自定义命令"标签页中:

1. 在输入框中输入命令
2. 在"命令名称"字段中输入一个描述性名称
3. 点击"保存命令"按钮
4. 保存的命令将显示为按钮，点击即可快速使用
5. 要删除命令，输入命令名称后点击"删除命令"按钮

## 导入和导出命令

在"自定义命令"标签页中，您可以:

1. 点击"导出命令"按钮将所有自定义命令导出为JSON文件
2. 点击"导入命令"按钮从JSON文件导入命令，可选择合并或替换现有命令

## 主题切换

可以通过菜单栏中的"File > Theme"选择暗色或亮色主题。

## 提交反馈

如果您有任何建议或发现了问题:

1. 点击菜单栏中的"Help > Send Feedback"
2. 填写反馈表单，包括您的姓名、电子邮件、反馈类型和具体内容
3. 为应用程序评分(1-5星)
4. 点击"OK"提交

## 系统要求

- Python 3.6+
- PySide6 6.5.0+
- 支持Windows、macOS和Linux

## 配置文件

自定义命令和设置会自动保存在以下位置:
- Windows: `%USERPROFILE%\.cmd_tool_gui\config.json`
- macOS/Linux: `~/.cmd_tool_gui/config.json`

## 扩展

要添加新的命令类别:

1. 创建一个继承自`CommandWidget`的新类
2. 在`initializeCommands`方法中添加该类别的预定义命令
3. 在`MainWindow`类的`setupUI`方法中创建新标签页并添加到标签页组件中
